# SaaS Base

A modern, modular FastAPI backend powered by:
- 🔧 [FastAPI](https://fastapi.tiangolo.com/)
- 📦 [`uv`](https://docs.astral.sh/uv/) for ultra-fast package and environment management
- 🔄 [Alembic](https://alembic.sqlalchemy.org/) for database migrations
- ✅ `pytest` for testing

---

## 📁 Project Structure

```
.
├── app/               # Main application code
│   ├── \_api/          # API routes (v1)
│   ├── accounts/      # Account-related logic
│   ├── auth/          # Authentication logic (JWT, OAuth, etc.)
│   ├── core/          # Config, DB, and logging
│   ├── scripts/       # Data scripts and initializers
│   └── main.py        # App entry point
├── alembic/           # Alembic migrations
├── tests/             # Unit and integration tests
├── pyproject.toml     # Project config and dependencies
├── uv.lock            # Dependency lockfile (generated by uv)

````

---

## 🚀 Getting Started

### 1. Install [`uv`](https://docs.astral.sh/uv/)

```bash
pipx install uv
````

### 2. Set up the environment

```bash
uv venv
uv sync
uv compile      # (optional: generate lockfile)
```

### 3. Run the FastAPI server

```bash
uv run fastapi dev
```

Visit: [http://localhost:8000/docs](http://localhost:8000/docs)

---

## 🗃️ Database & Migrations

Make sure your `DATABASE_URL` is set (e.g. in `.env` or `app/core/config.py`).

### Create a migration:

```bash
alembic revision --autogenerate -m "Add new model"
```

### Apply migrations:

```bash
alembic upgrade head
```

---

## ✅ Running Tests

```bash
pytest
```

You can add fixtures and test configuration in the `tests/` folder.

---

## 🛠️ Development Tips

* Use `uv pip sync` to install from lockfile (like `poetry install`)
* Add optional dependencies with `uv pip install -d dev <package>`
* Format/lint with [`ruff`](https://docs.astral.sh/ruff/): `uv ruff check .`
